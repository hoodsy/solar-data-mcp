# solar-mcp-market

MCP server for US solar market intelligence: permitting/inspection/PTO timelines
(SolarTRACE), installed-system records (LBNL Tracking the Sun), utility-scale
facilities (USPVDB), and AHJ lookup (SunSpec registry). Bulk datasets sync into a
local DuckDB store with explicit `sync_*` tools and cited vintages.
Part of [solar-data-mcp](https://github.com/hoodsy/solar-mcp).

Quickstart lives in the repo root README.
