# solar-mcp-economics

MCP server answering "would solar pay off here?" with cited, decomposable numbers:
retail tariffs (OpenEI URDB), electricity prices (EIA v2), incentives (federal ITC +
DSIRE snapshots), and the composite `estimate_roi` with a full audit trail.
Part of [solar-data-mcp](https://github.com/loganbernard/solar-data-mcp).

Quickstart lives in the repo root README.
